工作区即目录
版本库 .git（暂存区stage 主分支master 主分支的指针HEAD）	
	git add a.txt 把a添加到暂存区
	git commit 把暂存区的内容提交到当前分支
	git checkout -- readme.txt 撤销工作区的修改
	git reset HEAD readme.txt 把暂存区修改撤销
	
	git remote add origin git@github.com:Mewell/123.git
	git push -u origin master
	git push origin master
	
	git checkout -b dev 创建并转到dev分支
	git checkout master	转到master分支
	git merge dev	合并dev到主分支 合并dev分支到当前分支（master或用git branch查看当前分支）
	git branch -d dev	删除dev分支
	git branch 查看当前分支
